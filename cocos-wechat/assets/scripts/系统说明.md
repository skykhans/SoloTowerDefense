# Cocos 脚本结构说明

## config

- `ChapterConfig.ts`：章节配置
- `TowerConfig.ts`：炮塔配置
- `EnemyConfig.ts`：敌人配置
- `WaveConfig.ts`：波次配置生成
- `LevelConfig.ts`：关卡配置
- `ConfigService.ts`：统一配置读取入口
- `ResourceManifest.ts`：场景、预制体和音频的统一资源清单
- `ResourceManifestValidator.ts`：资源清单静态校验工具

## core

- `GameState.ts`：全局状态
- `BattleTypes.ts`：战斗阶段类型定义
- `SceneNames.ts`：场景名称常量
- `SceneNavigator.ts`：场景跳转服务
- `BattleResultSession.ts`：跨场景结算会话
- `LocalStorageService.ts`：本地存档服务
- `SaveData.ts`：存档结构
- `StorageKeys.ts`：存档键名
- `StartupLogService.ts`：启动阶段统一日志与摘要输出
- `SceneBindingValidator.ts`：场景绑定校验工具
- `SceneStructureValidator.ts`：场景结构与节点命名校验工具
- `TutorialGuideService.ts`：新手引导文案与阶段状态入口
- `HomeProgressService.ts`：首页统计摘要、推荐关卡与按钮文案入口
- `HomeNoticeSession.ts`：结算返回首页时的临时提示会话
- `PrefabStructureValidator.ts`：预制体结构与命名校验工具

## entities

- `Enemy.ts`：敌人实体
- `EnemyView.ts`：敌人表现层，负责血条、类型外观、减速染色，以及护盾怪、分裂怪、治疗怪外观切换
- `Tower.ts`：炮塔实体
- `PrefabNodeNames.ts`：统一五类预制体的节点命名
- `PrefabStructureProfiles.ts`：统一五类预制体的关键结构节点清单
- `PrefabAssetProfiles.ts`：统一五类预制体的资源占位模板
- `TowerView.ts`：炮塔表现层，负责选中高亮与范围圈
- `Bullet.ts`：子弹实体
- `BulletView.ts`：子弹表现层，负责按炮塔类型切换外观
- `HitEffect.ts`：命中特效占位组件
- `BuildSpot.ts`：建造点实体
- `BuildSpotView.ts`：建造点表现层，负责可建造与占用状态显示

## map

- `PathManager.ts`：路径点管理
- `BuildSpotManager.ts`：建造点管理
- `BuildSpotClickHandler.ts`：建造点点击建塔入口

## systems

- `GameManager.ts`：游戏主流程入口
- `WaveSpawnSystem.ts`：波次时序刷怪
- `EnemyMovementSystem.ts`：敌人沿路径移动
- `TowerCombatSystem.ts`：炮塔索敌、发射与命中处理

## scenes

- `HomeSceneInstaller.ts`：首页场景接线与绑定校验
- `MainSceneInstaller.ts`：战斗场景接线与绑定校验
- `ResultSceneInstaller.ts`：结算场景接线与绑定校验

## platform

- `PlatformTypes.ts`：平台抽象类型定义
- `PlatformService.ts`：运行环境识别与统一平台入口
- `WeChatStorageAdapter.ts`：微信小游戏本地存储适配
- `WebStorageAdapter.ts`：浏览器本地存储适配
- `MemoryStorageAdapter.ts`：兜底内存存储适配
- `SystemInfoService.ts`：系统信息与安全区读取统一入口
- `VibrationService.ts`：震动能力统一入口
- `ShareService.ts`：分享能力统一入口
- `UILayoutService.ts`：界面布局模式与安全区内边距统一入口

## ui

- `HomeUI.ts`：首页与选关，含章节说明、进度汇总、推荐关卡与按钮状态反馈
- `BattleUI.ts`：战斗 HUD、阶段文案与按钮状态反馈
- `TowerPanel.ts`：炮塔详情、升级、出售与按钮状态反馈
- `PlatformDebugPanel.ts`：开发期平台与分辨率调试面板
- `AdaptiveLayoutGroup.ts`：统一处理布局容器的横竖排、间距和内边距切换
- `UILayoutController.ts`：统一处理紧凑布局缩放与横竖屏节点切换
- `UILayoutPresets.ts`：为首页、战斗、结算等常见场景提供布局默认参数
- `SceneLayoutProfiles.ts`：为三张主场景提供推荐挂载点与预设组合模板
- `SceneNodeNames.ts`：统一首页、战斗、结算场景的节点命名
- `SceneStructureProfiles.ts`：统一三张主场景的关键结构节点清单
- `SceneAssetProfiles.ts`：统一三张主场景的首版资源占位模板
- `SafeAreaPanel.ts`：界面安全区容器组件，基于 `Widget` 偏移应用安全区
- `ResultUI.ts`：结算展示、奖励结算、下一关入口、回首页提示与场景跳转
