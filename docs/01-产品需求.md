# SoloTowerDefense 产品需求简版

## 项目定位

SoloTowerDefense 是一个轻量、快节奏、低学习成本的塔防游戏原型，当前版本为 Web 原型，目标方向为后续迁移到微信小游戏。

## 目标用户

- 喜欢轻度策略与成长反馈的休闲玩家
- 使用碎片化时间体验单局 3 到 8 分钟小游戏的用户
- 对操作门槛敏感，更重视反馈和节奏感的微信小游戏用户

## 核心目标

- 做出一个可以快速上手的塔防战斗闭环
- 建立可扩展的塔、防线、敌人、波次与局内成长体系
- 为后续微信小游戏版本沉淀数值、玩法和 UI 结构

## 当前玩法闭环

1. 玩家选择塔类型
2. 点击建造位放置炮塔
3. 点击开始波次生成敌人
4. 炮塔自动攻击敌人
5. 玩家根据金币情况升级或出售炮塔
6. 玩家通过暂停、倍速、局内技能调整节奏
7. 若生命值清零则失败结算
8. 若继续守住波次则进入下一波

## 当前核心内容

- 塔类型
  - 速射塔：高攻速、低单发伤害
  - 重炮塔：低攻速、高单发伤害，附带小范围溅射
- 敌人类型
  - 普通敌人
  - 快速敌人
  - 重甲敌人
  - 每 5 波出现的 Boss
- 波次机制
  - 波数提升后敌人数量与强度增长
  - 左侧面板展示下一波预告与当前出兵进度
- 玩家操作
  - 建造、升级、出售
  - 暂停、2x 速度
  - 主动技能

## P0 必做功能

- 单局战斗闭环
- 两种以上炮塔
- 三种以上敌人
- 波次推进与 Boss 波
- 失败结算
- 基础音画反馈占位
- 本地记录或最优成绩
- 基础新手提示

## P1 建议功能

- 更多塔类型：减速塔、辅助塔、狙击塔、范围塔
- 更多敌人：护盾、分裂、治疗、召唤
- 关卡与章节
- 新手引导
- 任务与成就
- 局外成长
- 激励广告方案

## P2 扩展方向

- 每日挑战
- 无尽模式排行
- 活动关卡
- 皮肤和主题地图
- 云存档
- 社交与分享玩法
