# SoloTowerDefense 功能清单

## 当前 Web 原型已完成功能

### 基础战斗

- 敌人沿固定路径移动
- 敌人到达终点会扣除生命值
- 生命值归零后游戏结束
- 波次逐步提升强度
- 每 5 波触发 Boss 压力波

### 炮塔系统

- 速射塔
- 重炮塔
- 减速塔
- 建造
- 升级
- 出售
- 炮塔选中与属性显示
- 重炮塔溅射伤害
- 减速塔减速效果

### 敌人系统

- 普通敌人
- 快速敌人
- 重甲敌人
- Boss 敌人

### 玩家操作

- 开始波次
- 暂停 / 继续
- 2x 倍速
- 取消建造
- 快捷键操作

### 局内信息反馈

- 生命、金币、波次、击杀、场上敌人数
- 波次预告
- 出兵进度条
- 建造模式提示
- 炮塔详情面板
- 局内消息提示

### 局内技能与记录

- 空袭主动技能
- 技能冷却显示
- 本地最佳波次记录
- 失败结算面板

### 工程支持

- README 运行说明
- 主线一键启动脚本
- 旧版原型启动脚本已迁移到 `legacy-web/`
- 产品、技术、迁移和路线文档

## 当前 Cocos 微信小游戏骨架已完成

### 工程骨架

- `cocos-wechat/` 主目录
- `assets/scripts/config`
- `assets/scripts/core`
- `assets/scripts/entities`
- `assets/scripts/map`
- `assets/scripts/systems`

### 已实现脚本

- 炮塔配置
- 敌人配置
- 波次配置生成
- 关卡配置
- 配置统一读取入口 `ConfigService`
- 全局状态 `GameState`
- 战斗阶段类型 `BattleTypes`
- 场景导航 `SceneNavigator`
- 场景名常量 `SceneNames`
- 结算会话 `BattleResultSession`
- 本地存档 `LocalStorageService`
- 存档模型 `SaveData`
- 路径管理 `PathManager`
- 建造点管理 `BuildSpotManager`
- 建造点点击入口 `BuildSpotClickHandler`
- 实体脚本：敌人、炮塔、子弹、建造点
- 主流程入口 `GameManager`
- 波次时序系统 `WaveSpawnSystem`
- 敌人移动系统 `EnemyMovementSystem`
- 炮塔攻击系统 `TowerCombatSystem`
- UI 脚本：`HomeUI`、`BattleUI`、`TowerPanel`、`ResultUI`
- 基础新手引导状态与局内提示
- 章节配置与章节化关卡归属
- 炮塔选择、升级、出售
- 关卡星级评价
- 首次通关金币奖励
- 首页关卡通关状态与星级展示
- 首页总星级、已通关关卡数与推荐关卡展示
- 首页章节名与章节说明展示
- 结算返回首页后的即时提示反馈
- 结算返回首页后的关卡自动定位
- 结算页直接进入下一关
- 目标波次通关判定
- 首页关卡选择与存档联动
- `HomeScene / MainScene / ResultScene` 场景化架构

### 当前配置管理方式

- 无后端
- 无管理员后台
- 配置文件驱动
- 关卡和数值由 `config/` 目录统一管理
