# 目录结构说明

## 仓库根目录

```text
SoloTowerDefense/
  docs/
  cocos-wechat/
  legacy-web/
  tools/
  start.bat
  start.ps1
```

## 根目录说明

- `docs/`
  - 项目文档目录
  - 包含产品需求、技术选型、迁移方案、开发路线、当前状态、功能清单等
- `cocos-wechat/`
  - 当前主开发目录
  - 用于承载 `Cocos Creator + TypeScript + 微信小游戏` 版本
- `legacy-web/`
  - 旧版 Web 验证原型目录
  - 当前只保留作玩法和数值参考，不再作为主开发方向
- `start.bat / start.ps1`
  - 当前主线一键启动脚本
  - 优先打开 Cocos Creator 项目或关键接入文档
- `tools/`
  - 开发期辅助脚本目录

当前已提供：

- `check-resource-manifest.ps1`
  - 资源清单检查脚本

## `cocos-wechat/assets/scripts/` 分层说明

### `config/`

配置层，负责定义静态配置数据。

- `ChapterConfig.ts`
  - 章节配置
- `TowerConfig.ts`
  - 炮塔配置
- `EnemyConfig.ts`
  - 敌人配置
- `WaveConfig.ts`
  - 波次生成配置
- `LevelConfig.ts`
  - 关卡配置
- `ResourceManifest.ts`
  - 场景、预制体和音频的统一资源清单
- `ResourceManifestValidator.ts`
  - 资源清单静态校验工具

### `core/`

核心基础层，负责跨系统共享的数据与基础服务。

- `GameState.ts`
  - 全局运行时状态
- `SceneNames.ts`
  - 场景名称常量
- `SceneNavigator.ts`
  - 场景跳转服务
- `BattleResultSession.ts`
  - 跨场景战斗结果会话
- `SaveData.ts`
  - 本地存档结构
- `StorageKeys.ts`
  - 存档键名常量
- `LocalStorageService.ts`
  - 本地存储服务
- `StartupLogService.ts`
  - 启动阶段日志与摘要服务
- `SceneBindingValidator.ts`
  - 场景 Inspector 绑定校验工具
- `SceneStructureValidator.ts`
  - 场景结构与节点命名校验工具
- `TutorialGuideService.ts`
  - 新手引导文案与阶段状态入口
- `HomeProgressService.ts`
  - 首页进度汇总、推荐关卡与按钮文案计算
- `HomeNoticeSession.ts`
  - 结算返回首页时的临时提示会话
- `PrefabStructureValidator.ts`
  - 预制体结构与命名校验工具

### `entities/`

实体层，负责可实例化的游戏对象。

- `Tower.ts`
  - 炮塔实体
- `PrefabNodeNames.ts`
  - 预制体节点命名常量
- `PrefabStructureProfiles.ts`
  - 预制体关键结构模板
- `PrefabAssetProfiles.ts`
  - 预制体资源占位模板
- `TowerView.ts`
  - 炮塔表现组件
- `Enemy.ts`
  - 敌人实体
- `EnemyView.ts`
  - 敌人表现组件
- `Bullet.ts`
  - 子弹实体
- `BulletView.ts`
  - 子弹表现组件
- `HitEffect.ts`
  - 命中特效占位组件
- `BuildSpot.ts`
  - 建造点实体
- `BuildSpotView.ts`
  - 建造点表现组件

### `map/`

地图层，负责路径和建造位等地图信息。

- `PathManager.ts`
  - 路径点管理
- `BuildSpotManager.ts`
  - 建造点管理

### `platform/`

平台适配层，负责微信小游戏与 Web 环境差异收口。

- `PlatformTypes.ts`
  - 平台抽象类型
- `PlatformService.ts`
  - 平台识别与统一入口
- `WeChatStorageAdapter.ts`
  - 微信小游戏存储适配
- `WebStorageAdapter.ts`
  - 浏览器存储适配
- `MemoryStorageAdapter.ts`
  - 兜底内存存储适配
- `SystemInfoService.ts`
  - 系统信息读取入口
- `VibrationService.ts`
  - 震动能力入口
- `ShareService.ts`
  - 分享能力入口
- `UILayoutService.ts`
  - 布局模式与安全区占位入口

### `systems/`

系统层，负责驱动战斗流程与核心循环。

- `GameManager.ts`
  - 游戏主流程入口
- `WaveSpawnSystem.ts`
  - 波次时序刷怪
- `EnemyMovementSystem.ts`
  - 敌人沿路径移动
- `TowerCombatSystem.ts`
  - 炮塔索敌与命中处理

### `scenes/`

场景接线层，负责把场景节点、系统和 UI 依赖在入口处连接起来。

- `HomeSceneInstaller.ts`
  - 首页场景接线
- `MainSceneInstaller.ts`
  - 战斗场景接线
- `ResultSceneInstaller.ts`
  - 结算场景接线

### `ui/`

界面层，负责首页、战斗 HUD、炮塔面板、结算面板。

- `HomeUI.ts`
- `BattleUI.ts`
- `TowerPanel.ts`
- `PlatformDebugPanel.ts`
- `AdaptiveLayoutGroup.ts`
- `UILayoutController.ts`
- `UILayoutPresets.ts`
- `SceneLayoutProfiles.ts`
- `SceneNodeNames.ts`
- `SceneStructureProfiles.ts`
- `SafeAreaPanel.ts`
- `SceneAssetProfiles.ts`
- `ResultUI.ts`
- `界面接入说明.md`

说明：

- `HomeUI.ts` 已支持关卡按钮状态反馈
- `HomeUI.ts` 已支持首页进度汇总、推荐关卡与结算回显
- `ResultUI.ts` 已支持标题、提示文案、星级、奖励、下一关入口与回首页反馈
- `EnemyView.ts` 已支持护盾怪独立外观节点，缺省时回退到重甲外观

```text
cocos-wechat/
  assets/
    audio/
      README.md
    prefabs/
      README.md

      README.md
```

- `audio/`
  - 音频资源目录
- `prefabs/`
  - 敌人、炮塔、子弹、建造点与特效预制体
- `scenes/`
  - `HomeScene / MainScene / ResultScene` 等场景文件
- `textures/`
  - 图片资源目录

说明：

- 这四个目录已经在仓库中创建
- 当前先放置说明文件，后续由 Cocos Creator 生成实际资源元文件
