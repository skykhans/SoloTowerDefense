# 目录结构说明

## 仓库根目录

```text
SoloTowerDefense/
  .creator/
  assets/
  docs/
  legacy-web/
  profiles/
  settings/
  tools/
  .gitignore
  package.json
  start.bat
  start.ps1
  tsconfig.json
```

## 根目录说明

- `.creator/ assets/ profiles/ settings/ package.json / tsconfig.json`
  - 当前 Cocos Creator 主工程
  - 根目录就是微信小游戏主线，不再额外嵌套 `cocos-wechat/`
- `docs/`
  - 项目文档目录
  - 包含需求、技术选型、迁移方案、开发路线、结构说明和检查清单
- `legacy-web/`
  - 旧版 Web 验证原型
  - 当前只保留作玩法和数值参考，不再作为主开发方向
- `tools/`
  - 开发期辅助脚本目录
- `start.bat / start.ps1`
  - 当前主线一键启动脚本

## assets/scripts 分层说明

- `config/`
  - 静态配置数据
- `core/`
  - 全局状态、存档、场景导航、校验器等核心能力
- `entities/`
  - 炮塔、敌人、子弹、建造点及其表现组件
- `map/`
  - 路径点和建造位管理
- `platform/`
  - 微信小游戏与 Web 环境差异收口
- `scenes/`
  - 场景 Installer 与运行时 bootstrap
- `systems/`
  - 刷怪、移动、战斗、主流程系统
- `ui/`
  - 首页、战斗、结算与布局适配相关 UI 组件
