# 微信小游戏发布前检查清单

## 目标

本文档用于在准备构建和发布微信小游戏前，做一次最小闭环检查，降低“能开发不能发布”的风险。

## 一、场景与流程

- [ ] `HomeScene` 能正常进入
- [ ] `MainScene` 能正常进入
- [ ] `ResultScene` 能正常进入
- [ ] 首页进入战斗流程正常
- [ ] 战斗结束进入结算流程正常
- [ ] 结算页返回首页流程正常
- [ ] 结算页重开流程正常

## 二、核心玩法

- [ ] 能选择速射塔建造模式
- [ ] 能选择重炮塔建造模式
- [ ] 能选择减速塔建造模式
- [ ] 能取消建造模式
- [ ] 能在建造位成功建塔
- [ ] 炮塔能自动索敌
- [ ] 子弹能飞行并命中
- [ ] 命中后有基础反馈
- [ ] 敌人抵达终点会扣血
- [ ] 生命归零后会失败
- [ ] 达成目标波次后会胜利

## 三、UI 与交互

- [ ] 首页上一关按钮状态正确
- [ ] 首页下一关按钮状态正确
- [ ] 首页开始按钮状态正确
- [ ] 首页分享按钮可点击且提示正确
- [ ] 战斗页开始波次按钮状态正确
- [ ] 战斗页暂停按钮状态正确
- [ ] 炮塔面板未选中时按钮禁用
- [ ] 炮塔面板金币不足时升级按钮禁用
- [ ] 结算页标题和提示文案正确
- [ ] 结算页分享按钮可点击且提示正确

## 四、存档

- [ ] 最佳波次能保存
- [ ] 已解锁关卡能保存
- [ ] 当前选择关卡能保存
- [ ] 结算后返回首页能读到最新存档
- [ ] 重启后仍能读到本地存档

## 五、预制体与接线

- [ ] `Enemy.prefab` 已挂基础脚本
- [ ] `Tower.prefab` 已挂基础脚本
- [ ] `BuildSpot.prefab` 已挂基础脚本
- [ ] `Bullet.prefab` 已挂基础脚本
- [ ] `HitEffect.prefab` 已挂基础脚本
- [ ] `MainSceneInstaller` 校验通过
- [ ] `HomeSceneInstaller` 校验通过
- [ ] `ResultSceneInstaller` 校验通过

## 六、平台适配

- [ ] `PlatformService` 能正确识别微信小游戏环境
- [ ] Web 环境下能正确回退到浏览器存储
- [ ] 非浏览器环境下能正确回退到内存存储
- [ ] 业务层未直接散落调用 `wx` 本地存储
- [ ] `SystemInfoService` 能返回有效环境信息
- [ ] 微信环境下能读到有效 `safeArea` 数据
- [ ] `VibrationService` 在目标平台可安全调用
- [ ] `ShareService` 在目标平台可安全调用或优雅回退
- [ ] 启动日志能输出平台、存储模式与存档摘要
- [ ] `UILayoutService` 能返回有效布局模式
- [ ] `SafeAreaPanel` 仅通过 `Widget` 偏移适配安全区，不直接改容器尺寸
- [ ] 如启用 `UILayoutController`，紧凑布局缩放与横竖屏节点显隐切换正确
- [ ] 如启用 `AdaptiveLayoutGroup`，横竖屏切换时按钮组方向、间距与 padding 正确
- [ ] 首页 `LevelPanel / ActionPanel` 在横竖屏切换时无重叠
- [ ] 战斗页 `ActionButtons / BuildToolbar` 在紧凑布局下仍可点击
- [ ] 结算页按钮区在小屏设备下无遮挡
- [ ] 建塔成功后短震动表现正常
- [ ] 升级成功后短震动表现正常
- [ ] 失败结算时长震动表现正常
- [ ] 如启用调试面板，平台与分辨率显示正确
- [ ] 如启用调试面板，存储模式显示正确
- [ ] 如启用安全区容器，核心 HUD 未被边缘裁切
