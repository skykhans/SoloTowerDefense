# 场景节点命名约定

## 目标

本文档统一 `HomeScene / MainScene / ResultScene` 的推荐节点命名，避免：

- 文档里一个名字
- Cocos 场景里另一个名字
- 后续调试和排查再出现第三套名字

代码常量位置：

- [`SceneNodeNames.ts`](/C:/Test/SoloTowerDefense/cocos-wechat/assets/scripts/ui/SceneNodeNames.ts)

## 当前原则

- 优先沿用常量里已有的命名
- 新增节点如果属于某一块功能区，尽量保持同一风格
- 先补常量，再补文档，再进场景里创建节点
