# 最小可运行资源清单

## 目标

本文档整理当前项目在进入 Cocos Creator 后，建议优先准备的最小资源集合，用于尽快搭出可运行的第一版：

- 三张主场景
- 五类核心预制体
- 基础贴图占位
- 基础音频占位

## 二、预制体资源

相关代码：

- [`PrefabAssetProfiles.ts`](../assets/scripts/entities/PrefabAssetProfiles.ts)
- [`ResourceManifest.ts`](../assets/scripts/config/ResourceManifest.ts)

推荐先准备：

- 敌人四类占位图 + 血条填充图
- 塔底座、三类塔身、选中圈、范围圈
- 建造点空闲、占用、提示图
- 三类子弹占位图
- 命中特效核心图

## 三、音频资源

当前前期建议先准备：

- `build.wav`
- `upgrade.wav`
- `hit.wav`
- `enemy-death.wav`
- `victory.wav`
- `failure.wav`

说明：

- 文件名只是建议，不要求现在就接播放逻辑
- 先放占位音频，后续再统一接音频服务
- 统一读取入口可走 `ConfigService.getAudioAssetProfile()`

## 四、推荐优先级

1. 先准备 `HomeScene / MainScene / ResultScene` 面板和按钮占位图
2. 再准备 `Enemy / Tower / BuildSpot` 三类预制体占位图
3. 再补 `Bullet / HitEffect`
4. 最后补音频占位

## 五、当前目标

最小目标不是“美术完整”，而是：

- 场景能打开
- 预制体能实例化
- 结构自检通过
- 主流程能跑通
- 首次进入时能看到基础引导提示
